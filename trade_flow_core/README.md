# TradeFlow Core

TradeFlow Core is the backbone of the TradeFlow platform, including Rust libraries for event-driven trading, reinforcement learning, and real-time agents. This program is ideal for quantitative traders due to its dependability, performance, modularity, testability, maintenance, and ease of deployment.
